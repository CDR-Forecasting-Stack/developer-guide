# CDR Forecasting Stack Developer Guide

This website is for developers of the CDR Forecasting Stack. 
Its purpose is to help on-board new members.


## Modifying the site

The site if built using [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/)


```sh title="serve the site locally"
uv run mkdocs serve --livereload
```
